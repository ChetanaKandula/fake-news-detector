# 📰 Fake News Detector (AI-Powered)

A full-stack web application that detects whether a given news article is **Fake or Real** using AI (NLP models).

---

## 🚀 Features

- 🧠 AI-based fake news detection using Hugging Face
- 🌐 Full-stack application (Frontend + Backend)
- ⚡ Real-time prediction with API integration
- 🎨 Clean and responsive user interface
- 🔄 Handles loading and error states

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### AI / API
- Hugging Face Transformers API (`facebook/bart-large-mnli`)
